Start/Stop SEO-Optimizer

SSH-Verbindung & App-Neustart Befehle für deinen VM Server

VM Verbindungsdaten anpassen (für dynamische Befehle)
1. SSH Login CMD / PowerShell / Terminal
Gib diesen Befehl in deiner Eingabeaufforderung (CMD) oder PowerShell ein, um dich mit deinem Server zu verbinden.
ssh root@217.160.62.38
2. Neustart via Systemd (Empfohlen) Dienst / Service
Verwende diesen Befehl, wenn das Tool als systemweiter Hintergrunddienst auf dem Server eingerichtet ist.
sudo systemctl restart seo-optimizer
3. Manueller Prozess-Neustart (nohup / screen) Script / Hintergrundprozess
Verwende diese Befehle nacheinander, um einen direkt ausgeführten Python-Prozess zu finden, zu beenden und im Hintergrund neu zu starten.
A. Laufenden Prozess suchen:
ps aux | grep seo-optimizer
B. Laufenden Prozess beenden:
pkill -f seo-optimizer
C. In den Projektordner wechseln:
cd /var/www/seo-optimizer
D. Applikation im Hintergrund neu starten (Port 8001):
nohup venv/bin/uvicorn main:app --host 0.0.0.0 --port 8001 > uvicorn.log 2>&1 &
